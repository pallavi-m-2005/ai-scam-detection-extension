# AI Scam Detection Extension

An AI-powered browser extension that detects phishing and scam websites using:

- URL analysis
- ML prediction
- Screenshot similarity AI
- Browser behavior analysis
- Risk scoring

## Features

✅ URL scanning  
✅ Popup warning  
✅ Phishing detection  
✅ Screenshot AI  
✅ History analytics  

## Tech Stack

- JavaScript
- Flask
- Python
- Scikit-learn
- OpenCV

## Future Scope

- Real-time cloud detection
- Voice scam detection
- QR phishing detection
